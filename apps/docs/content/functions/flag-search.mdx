# Infection

## Request Parameters

### Native Parameters
| Name | Description | Required | Array |
|------|-------------|----------|--------|
| category | Always "infection" | Yes | No |
| encounter | Associated encounter | No | No |
| patient | The associated patient | Conditional | No |
| status |  | No | No |
| subject | The associated patient | Conditional | No |

### Post-filter Parameters
| Name | Description | Required | Array |
|------|-------------|----------|--------|
| date | The period during which the infection was marked as active | No | No |
