## Set up Epic integration

Follow these steps to set up your Epic integration:

### 1. Create a FHIR on Epic account

At the moment, we are abstracting over the FHIR on Epic APIs. [Go to FHIR on Epic](https://fhir.epic.com/) and make an account. 


### 2. Create a new application
1. Navigate to the "Build Apps" tab
2. Click the "+Create" button

<Frame>
  <img src="/EpicSteps/step1.png" alt="Create new Epic application" />
</Frame>

### 3. Configure your application
1. Enter your application name
2. For Application Audience, select 'Clinicians or Administrative Users'
3. Select the APIs you want to use
   <Callout>We recommend selecting all APIs initially for development purposes. Note that this may take a minute to complete.</Callout>

<Frame>
  <img src="/EpicSteps/step2.png" alt="Configure Epic application" />
</Frame>

### 4. Set up redirect URIs
Add your redirect URIs for both:
- Development domain (e.g., `http://localhost:3000`)
- Production URL

<Frame>
  <img src="/EpicSteps/step3.png" alt="Set up redirect URIs" />
</Frame>

<Callout> You do not need to fill out any of these </Callout>

<Frame>
  <img src="/EpicSteps/step4.png" alt="Set up redirect URIs" />
</Frame>

### 5. Complete application setup
1. Fill out the remaining information
2. Click "Save & Ready for Production"

<Frame>
  <img src="/EpicSteps/step5.png" alt="Set up redirect URIs" />
</Frame>
