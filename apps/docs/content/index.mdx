## Introduction

Nerve is a unified API for healthtech developers to easily integrate with provider's EHR systems, and build using sandbox data. We
support major platforms like Epic, Cerner, and AthenaHealth and [many more](/docs/integrations)!

Nerve is built on top of the [FHIR](https://hl7.org/fhir/) data model, and offers developers consistent abstractions
for reading and writing data to EHR systems.

## Quick Setup

1. **Get your sandbox API Key**

   Request your sandbox API key from the [Developer Portal](https://nerve.run/dashboard).

2. **Install SDK**

   ```package-install
   npm install @nerve-js/client
   ```

## Basic Usage

<Tabs defaultValue="vanilla">
  <Tab label="Vanilla" value="vanilla">
    ```typescript
    import { NerveClient } from '@nerve-js/client';

    const client = new NerveClient({
      redirectUrl: 'https://your-app.com/callback',
      scope: ['patient.read', 'patient.write']
    });

    // Search for a patient
    const results = await client.patient.search("John Doe");
    ```
  </Tab>
  <Tab label="Tanstack" value="tanstack">
    ```typescript
    import { NerveClient } from '@nerve-js/client';
    import { useQuery } from '@tanstack/react-query';

    const client = new NerveClient({
      redirectUrl: 'https://your-app.com/callback',
      scope: ['patient.read', 'patient.write']
    });

    function YourComponent() {
      const { data: results } = useQuery({
        queryKey: ['patient-search'],
        queryFn: () => client.patient.search("John Doe")
      });
    }
    ```
  </Tab>
</Tabs>
